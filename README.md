# 🧙 Harry Potter Web

**Harry Potter Web** is a complete, interactive world for fans of the wizarding world. Built with deep magical aesthetics, including custom typography and glowing glassmorphism character cards, this site provides an immersive experience.

---

## ✨ Magical Features

- **Magical Glassmorphism UI**: High-end translucent elements designed to evoke the mystery of the wizarding world.
- **Glowing Character Cards**: Interactive character panels with glowing hover effects.
- **Custom Movie-Themed Typography**: Using the "Cinzel" font to maintain cinematic authenticity.
- **Netlify Integrated**: Ready-to-go hosting for global fan access.
